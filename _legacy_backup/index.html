Setup Guide

  1. Open your Google Spreadsheet.
  2. Go to Extensions > Apps Script.
  3. Copy contents of google_apps_script.js from the project folder and paste them there. Save it.
  4. Click Deploy > New deployment. Select Web app.
  5. Configure Who has access as Anyone and click Deploy.
  6. Copy the Web App URL and paste it in the field above.
